## üè¶ Overview

EMIPredict AI is a comprehensive financial risk assessment platform that leverages machine learning to provide intelligent EMI (Equated Monthly Installment) prediction and eligibility assessment. Built for financial institutions, FinTech companies, and banks, this platform processes 400,000+ financial records to deliver accurate, real-time risk assessments.

### Key Features
- **Dual ML Models**: Classification for EMI eligibility (3 classes) and Regression for maximum EMI amount
- **Real-time Predictions**: Instant risk assessment for loan applications
- **MLflow Integration**: Comprehensive experiment tracking and model registry
- **Interactive Dashboard**: Multi-page Streamlit application with advanced visualizations
- **Data Management**: Complete CRUD operations for financial data
- **Scalable Architecture**: Production-ready deployment on Streamlit Cloud
